PA Assistant Setup Guide

Welcome to PA Assistant! Before scanning specific Drive folders, you must execute a one-time configuration mapping securely tying your Workspace environment into Google Cloud APIs.

Phase 1: Create a Google Cloud Project

  1. Go to console.cloud.google.com.
  2. Click New Project. Name it "PA Assistant Cloud" and create it.
  3. Go to APIs & Services → Library.
  4. Enable the Google Drive API, Google Drive Activity API, and Google Picker API.

Phase 2: Link Apps Script to Cloud

  1. In Google Cloud Console, click Project settings (three dots top right). Copy the "Project number".
  2. In your Apps Script Editor, click Project Settings (⚙️).
  3. Under "Google Cloud Platform (GCP) Project", click Change project.
  4. Paste your Project number to link them.

Phase 3: Get a Picker API Key

  1. In Google Cloud Console, go to APIs & Services → Credentials.
  2. Click Create Credentials → API Key.
  3. Copy the string provided.

Phase 4: Set the Developer Key

  1. In Apps Script Editor, click Project Settings (⚙️).
  2. Scroll to Script Properties and click Add script property.
  3. Set the property name to exactly: DEVELOPER_KEY
  4. Paste the API Key from Phase 3 into the Value box and save.

Team Sharing

  1. Your team does not need to repeat these steps!
  2. In Apps Script Editor, click Share (top right).
  3. Add your colleagues' Google emails as Editor or Viewer.
  4. They can now open the Sidebar natively from Google Docs!
