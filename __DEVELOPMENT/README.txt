# Personal Work Summarizer

Generated by Multi-Agent Pipeline

## Documents

| File | Description |
|------|-------------|
| `01-vision-document.md` | Product vision, value proposition, and scope |
| `02-research-report.md` | API research and integration findings |
| `03-prd-document.md` | Product requirements and specifications |
| `main.py` | Python implementation |

## Quick Start

```bash
# Install dependencies
pip install google-generativeai httpx google-auth-oauthlib google-api-python-client keyring

# Run the summarizer
python main.py --start-date 2024-03-01 --end-date 2024-03-15 --output summary.md
```

## Configuration

Set environment variables:
```bash
export GEMINI_API_KEY="your-gemini-key"
export PERPLEXITY_API_KEY="your-perplexity-key"
```

## Output Format

The tool generates a Markdown table with columns:
- Date/Time
- Platform
- Work Done
- File Links
- Comments
